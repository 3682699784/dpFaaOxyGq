# 项目简介

## 核心功能
- 🩺 医疗问答
- 💬 实时对话
- 🧠 上下文记忆
- 📚 知识检索
- 🎨 美观界面
- 🔄 对话记忆
- 🔄 个人中心功能
- 🔄 历史对话功能
- 🔄 登录注册功能
## 技术栈

### 前端
- **框架**：React 19 + TypeScript
- **状态管理**：React Hooks
- **样式**：CSS Modules
- **网络请求**：Fetch API（支持流式响应）
- **组件**：
  - 自定义聊天界面
  - Markdown渲染 (react-markdown)
  - 弹窗组件
  - 消息提示 (react-hot-toast)

### 后端
- **框架**：FastAPI (Python)
- **AI框架**：LangChain 0.3.0
- **大语言模型**：通义千问 (qwen-turbo/qwen-plus/qwen-max)
- **向量数据库**：Chroma
- **文本嵌入**：DashScope Embeddings
- **流式响应**：SSE (Server-Sent Events)
- **文档处理**：LangChain Text Splitters

## 适用场景
- 提供便捷的医疗咨询服务
- 支持用户进行自我健康管理
- 适用于医院、诊所等医疗机构的辅助问诊
- 适用于个人用户进行健康问题咨询
## 技术特点
- 采用前后端分离架构，前端React，后端FastAPI
- 利用LangChain框架和通义千问大语言模型实现智能问答
- 支持Markdown渲染，提供现代化的聊天UI
